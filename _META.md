# Productivity Calculator - 项目根目录

一个基于 Next.js 的 pSEO 生产力计算器工具站，自动生成 20+ 行业专属计算器页面。

## 文件清单

| 文件/目录 | 地位/作用 |
|-----------|----------|
| `src/` | 源代码目录，包含所有应用代码 |
| `public/` | 静态资源目录 |
| `PROJECT_RULES.md` | 项目开发规范 - 分形文档结构规则 |
| `package.json` | 依赖管理和脚本配置 |
| `next.config.ts` | Next.js 框架配置 |
| `tsconfig.json` | TypeScript 配置 |
| `tailwind.config.ts` | Tailwind CSS 配置 |
| `postcss.config.mjs` | PostCSS 配置 |
| `eslint.config.mjs` | ESLint 代码规范配置 |
